# 🍽️ Swiggy Food Delivery Analysis Dashboard (Excel Project)

## 📌 Project Overview
This project analyzes Swiggy food delivery data using Microsoft Excel.
The objective is to understand sales performance, customer ratings,
food preferences, and regional business trends through an interactive dashboard.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning
- Data Visualization

---

## 📊 Key Performance Indicators (KPIs)

### 1. Total Sales (₹)
Calculated the overall revenue generated from food orders.

### 2. Average Rating
Analyzed customer satisfaction levels across restaurants.

### 3. Average Order Value
Measured average revenue generated per order.

### 4. Ratings Count
Calculated total customer reviews.

### 5. Total Orders
Analyzed total number of food orders received.

---

## 📈 Dashboard Insights

### Monthly Sales Trend
Analyzed month-wise sales performance and revenue changes.

### Daily Sales Trend
Tracked daily order and revenue variations.

### Total Sales by Food Type
Compared Veg vs Non-Veg revenue contribution.

### State Wise Sales Analysis
Visualized sales distribution across different states.

### Quarterly Performance Analysis
Compared Sales, Ratings, and Orders by quarter.

### Top 5 Cities by Sales
Identified highest revenue generating cities.

### Weekly Trend Analysis
Monitored weekly sales patterns and peak periods.

---

## 🔍 Business Insights

- Identified top performing cities based on sales.
- Compared customer preferences by food category.
- Analyzed sales trends over time.
- Measured customer satisfaction through ratings.
